安装centos

CentOS（Community Enterprise Operating System，中文意思是：社区企业操作系统）是Linux发行版之一，它是来自于Red Hat Enterprise Linux依照开放源代码规定释出的源代码所编译而成。由于出自同样的源代码，因此有些要求高度稳定性的服务器以CentOS替代商业版的Red Hat Enterprise Linux使用。两者的不同，在于CentOS并不包含封闭源代码软件。

可以到Centos官网 下载最新版本

安装和使用git

在github网站上搭建blog,我们既拥有绝对管理权，又享受github带来的便利----不管何时何地，只要向主机提交commit，就能发布新文章。更妙的是，这一切还是免费的，github提供无限流量，世界各地都有理想的访问速度。

详细的git命令教程可以参见廖雪峰git教程

我们使用下面git命令来复制一个github的repository到本机 


$ git clone https://github.com/username/username.github.io

安装和使用jekyll

Jekyll是一个静态站点生成器，它会根据网页源码生成静态文件。它提供了模板、变量、插件等功能，所以实际上可以用来编写整个网站。先在本地编写符合Jekyll规范的网站源码，然后上传到github，由github生成并托管整个网站。

当然，我们先要在本地安装Jekyll，测试运行情况，再更新到github

1.安装ruby环境

可以到ruby官网下载最新版本, 安装完成后使用下面命令可以看到ruby版本 


$ ruby --version

2.安装bundle 


$ gem install bundler

3.在git工程下创建一个名为Gemfile的文件在里面写入如下内容： 


source 'https://rubygems.org'
gem 'github-pages'

最后在git目录下运行jekyll 


$ bundle exec jekyll serve

生成博客页面
